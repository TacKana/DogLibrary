狗库是基于AI生成答案的智能题库系统，提供标准化API接口，适配各类自动答题插件。通过联网搜索功能，大幅提升AI答题准确率。

## 核心功能

### 🤖 AI题库
- 使用定制化提示词生成标准答案，兼容答题插件
- 缓存历史题目，避免重复生成
- 支持所有OpenAI对话接口
- 可自定义端口并开放局域网访问

### 📖 本地题库管理
- 提供完整的缓存题目管理功能

### 🔎 联网搜索（待开发）
- 类似于各家ai内置的搜索功能，用于解决AI答案时效性问题和准确率问题

## 技术特点

- **跨平台支持**：兼容Windows、Linux、macOS等主流操作系统
- **隐私保护**：除AI平台接口外，无任何第三方数据调用
- **智能搜索**：联网搜索功能显著提升答题准确率

## 为什么选择狗库？

1. **简单易用**：图形化界面操作，详细教程文档
2. **功能强大**：支持联网搜索、题库缓存等核心功能
3. **持续更新**：定期发布新功能和性能优化

# doglibrary

一个使用 Electron、Vue 和 TypeScript 构建的应用程序

## 推荐的 IDE 设置

- [VSCode](https://code.visualstudio.com/) + [ESLint](https://marketplace.visualstudio.com/items?itemName=dbaeumer.vscode-eslint) + [Prettier](https://marketplace.visualstudio.com/items?itemName=esbenp.prettier-vscode) + [Volar](https://marketplace.visualstudio.com/items?itemName=Vue.volar)

## 项目设置

### 安装

```bash
$ npm install
```

### 开发

```bash
$ npm run dev
```

### 构建

```bash
# Windows 系统
$ npm run build:win

# macOS 系统
$ npm run build:mac

# Linux 系统
$ npm run build:linux
```
